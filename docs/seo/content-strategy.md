# SEO Content Framework (English / Ethiopia)

## Primary objective
Help users find this site when searching for worship learning content and worship ministry topics in Ethiopia.

## Keyword clusters and page mapping
- **How-to singing intent**: target with blog tutorials (voice warmups, pitch control, breathing, harmony practice).
- **Worship ministry intent**: target with pillar blog pages and courses/teaching pages (team leadership, planning, biblical foundations).
- **Church team intent**: target with practical workflow content (rehearsal structure, service prep, volunteer onboarding).

## Recommended cornerstone pages
1. `How to Sing Better for Worship (Beginner Guide)`
2. `How to Start and Grow a Worship Ministry in Ethiopia`
3. `Weekly Worship Team Rehearsal Plan for Churches`

## Blog publishing template
Use this checklist for every new post:
- Primary keyword appears in title, slug, first paragraph, one H2, and meta description.
- Include 3-5 H2 sections that answer clear user questions.
- Add one internal link to `/courses` and one internal link to another relevant blog post.
- Add one external authority citation when useful (music theory, vocal health, ministry references).
- Add an FAQ section with 2-4 short Q/A items for long-tail search queries.

## Internal linking rules
- Every blog post must link to at least one relevant teaching page or `/courses`.
- Every teaching detail page should link back to `/blog` and one relevant article when available.
- Homepage sections should continue linking to newest blog posts and latest teachings.

## Metadata writing pattern
- Keep title around 50-60 characters.
- Keep meta description around 140-160 characters.
- Use Ethiopia context naturally when relevant (not keyword stuffing).
